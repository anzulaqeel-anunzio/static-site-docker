# Single-File Static Site Dockerfile

The smallest, fastest way to dockerize a static HTML website using Nginx Alpine.

## Features

-   **Extremely Lightweight**: ~10MB image size.
-   **High Performance**: Nginx optimized for static content.
-   **Simple**: Just copy HTML files and run.

## Installation

1.  **Clone the repository**.
2.  **Build and Run**:
    ```bash
    docker build -t my-website .
    docker run -p 80:80 my-website
    ```

## Usage

Replace the contents of `public/` with your own HTML/CSS/JS.
